Introducing Pico.

A stupidly simple & blazing fast, flat file CMS.
Making the web easy.

DOWNLOAD NOW

Pico makes creating and maintaing a website
as simple as editing text files.		 Pico is seriously lightweight and doesn't use a
database, making it super fast.		 Pice is a "flat file" CMS, meaning no database woes,
no MySQL queries, nothing.
Edit your website in your favorite text editor using
simple Markdown formatting		 Pico uses the Twig templating engine, for powerful
and flexible themes		 Pico is completely free and open source, released
under the MIT license
